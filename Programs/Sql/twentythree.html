Deleting Views
        Here's how you can delete a view in MySQL:
        
        Syntax
        DROP VIEW view_name;
        
        Example View : failStudents
        RollNo	Name	PhoneNo	Class	Marks	City
        4204	Asha	6259927794	BCA-I	156	Haryana
        Query
        DROP VIEW failStudents;
        
        Query OK, 0 rows affected (0.01 sec)
